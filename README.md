# AI Text Summarizer

A simple AI-powered text summarizer built with Python and Streamlit.

## Features
- Paste text
- Generate summary
- Simple web UI

## Tech Stack
- Python
- Streamlit
- GitHub

## Run locally
1. Clone the repo
2. Create a virtual environment
3. Install dependencies
4. Run the app

## Commands
pip install -r requirements.txt
streamlit run app.py
